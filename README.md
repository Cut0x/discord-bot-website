# Discord Bot Website
Template open source, sobre et responsive pour présenter votre bot Discord.

## Prérequis
- PHP (via XAMPP ou autre serveur local)

## Installation
1. Forkez ce dépôt pour l’utiliser comme base (c’est la meilleure option pour garder vos modifications).
2. Clonez votre fork en local.
3. Renommez `config.example.php` en `config.php`.
4. Renseignez au minimum :
   - `discord.bot_id`
   - `discord.bot_token`

## Démarrage
- Ouvrez `index.php` via votre serveur local (XAMPP).

## En cas d’erreur
Si une erreur est affichée, suivez les détails indiqués sur la page d’erreur.
Si le problème persiste, contactez-moi : https://valloic.fr/contact

## Points forts
- Design sobre, moderne et responsive
- Branding automatique depuis l’API Discord (nom, avatar, favicon, title)
- Configuration simple et rapide
- Sections prêtes à l’emploi (features, commandes, FAQ, avis)

## Sécurité
Le token du bot est utilisé côté serveur uniquement.
Ne jamais exposer le token dans le navigateur.
